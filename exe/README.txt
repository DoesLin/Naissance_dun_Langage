                README
======================================

config/ contient les fichiers :
	- config.xml, qui configure le système et ses individus
	- matrice.txt, qui décrit le voisinage du système
	- schema.xsd, décrivant la structure de config.xml

Naissance_dun_langage.jar requiert config/config.xml et 
config/matrice.txt correctement formatés pour fonctionner

Aucun argument de ligne de commande particulier n'est considéré
